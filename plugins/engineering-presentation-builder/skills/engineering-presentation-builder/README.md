# Engineering Presentation Builder Skill

Este diretório contém uma skill especializada, projetada para transformar temas técnicos em narrativas de alto impacto para apresentações e entrevistas de nível Especialista (Staff/Principal).

## 🚀 O que é?
Diferente de um gerador de texto comum, esta skill atua como um **mentor de entrevista**. Ela utiliza um processo de "Mineração de Contexto" através de perguntas estruturadas para extrair detalhes que muitas vezes esquecemos de mencionar, garantindo que sua apresentação demonstre senioridade sistêmica.

## 🧠 Frameworks Utilizados
A skill baseia-se em quatro pilares fundamentais:
- **STARL:** Situation, Task, Action, Result, Learning (o esqueleto da narrativa).
- **4D:** Drivers, Design, Data, Desdobramento (para decisões de arquitetura).
- **Impacto em 3 Níveis:** Técnico, Time e Negócio.
- **Evolução de Senioridade:** Transição do impacto "Eu" -> "Nós" -> "Sistema".

## 🛠 Como Usar
1.  **Ativação:** Após instalar, certifique-se de recarregar as skills com `/skills reload`.
2.  **Início:** Forneça um tema genérico, por exemplo:
    > "Preciso montar uma apresentação sobre a migração do sistema de mensageria."
3.  **Processo:** O agente fará perguntas específicas para cada fase (Contexto, Ação, Resultado, Aprendizado). **Responda com o máximo de detalhes que lembrar.**
4.  **Entrega:** Ao final, você receberá um roteiro estruturado pronto para ser transposto para slides ou usado como roteiro de fala.
