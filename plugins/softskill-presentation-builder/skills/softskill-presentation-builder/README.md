# Soft Skill Presentation Builder Skill

Este diretório contém uma skill especializada, projetada para transformar trajetórias pessoais e experiências comportamentais em narrativas de alto impacto para apresentações e entrevistas de nível Especialista (Staff/Principal).

## 🚀 O que é?
Diferente de um gerador de texto comum, esta skill atua como um **mentor de maturidade profissional**. Ela utiliza um processo de "Mineração de Contexto" através de perguntas estruturadas para extrair detalhes sobre sua evolução, comportamento e liderança, garantindo que você demonstre a senioridade exigida em papéis de Especialista.

## 🧠 Frameworks Utilizados
A skill baseia-se em três pilares fundamentais de soft skills:
- **HERO:** História, Evolução, Relevância, Objetivo (para identidade e motivação).
- **CORE:** Contexto, Obstáculo, Resposta, Evolução (para conflitos e relações humanas).
- **SPIRE:** Story, Padrões, Impacto, Reflexão, Evolução (para trajetória e estilo de liderança).

## 🛠 Como Usar
1.  **Ativação:** Após instalar, certifique-se de recarregar as skills com `/skills reload`.
2.  **Início:** Forneça um objetivo ou tema genérico, por exemplo:
    > "Preciso me preparar para a pergunta 'fale sobre você' para uma vaga de Staff."
    > "Quero estruturar um exemplo de como lido com feedbacks difíceis."
3.  **Processo:** O agente escolherá o framework mais adequado (HERO, CORE ou SPIRE) e fará perguntas específicas para cada fase. **Responda com o máximo de detalhes que lembrar.**
4.  **Entrega:** Ao final, você receberá um roteiro estruturado com pitch, narrativa organizada e frases de alto impacto.

## 📂 Estrutura
- `SKILL.md`: Instruções de comportamento para o Agente.
- `references/frameworks.md`: Base de conhecimento com o Kit Definitivo de Soft Skills.
